# 编程规范（参照阿里巴巴Java开发手册）

1. 缩进：统一使用4个空格
2. 变量命名：小写字母+下划线，如 file_path、capacity
3. 类命名：大驼峰，如 DataReader、MainUI
4. 函数命名：小写+下划线，如 read_d01kp_data、solve
5. 每行最大字符数：不超过80
6. 函数最大行数：不超过50行
7. 常量：全大写+下划线
8. 空行：类之间空2行，方法之间空1行
9. 注释：类、关键函数必须加注释，核心逻辑加行注释
10. 操作符空格：= + - * / > < 等前后必须加空格
11. 禁止使用无意义变量名（a/b/c）及拼音命名
