D01KP 背包问题求解项目
## 项目简介
本项目是基于 Python 开发的 D{0-1} 背包问题（D{0-1}KP）求解系统，采用动态规划算法，严格遵循制定的编程规范开发，实现了从数据读取、数据可视化到最优解求解、结果保存的全流程功能，适用于 D{0-1} 背包问题的学习与测试。

## 核心功能（按操作流程排序）
1.  数据读取功能：支持读取 TXT 格式的 D{0-1}KP 测试数据，自动解析背包容量、项集数量及每个项集的 3 个物品（重量、价值）信息，具备文件异常处理能力，避免因文件不存在、格式错误导致程序崩溃。
2.  数据可视化功能：基于读取的项集数据，绘制重量-价值散点图，横轴为物品重量，纵轴为物品价值，支持中文显示，添加网格线，直观展示所有物品的重量与价值对应关系。
3.  项集排序功能：按每个项集第三项物品的价值/重量比（价值密度）进行非递增排序，排序后实时提示排序完成，方便用户对比排序前后的项集差异。
4.  最优解求解功能：采用动态规划算法求解 D{0-1}KP 问题，在不超过背包容量的限制下，计算装入物品的最大价值，并记录求解耗时，结果实时展示在界面中。
5.  结果保存功能：支持将求解结果（背包容量、最优价值、求解耗时）保存为 TXT 文本文件或 Excel 表格文件，用户可自主选择保存路径，保存成功后弹出提示信息。

## 运行环境
编程语言：Python 3.8 及以上版本
依赖库：matplotlib（绘图）、pandas（Excel 保存）、tkinter（图形界面，Python 自带，无需额外安装）
依赖安装命令：pip install matplotlib pandas openpyxl

## 运行方式（步骤清晰，可直接操作）
1.  克隆本 GitHub 仓库到本地，或直接获取项目完整文件夹 `D01KP_Project`；
2.  打开终端（CMD、PowerShell 或 Git Bash），进入项目根目录（命令：cd 项目所在路径/D01KP_Project）；
3.  执行运行命令：python src/main.py；
4.  程序启动后，按界面按钮顺序操作（1.读取数据 → 2.绘制散点图 → 3.项集排序 → 4.求解最优解 → 5.保存结果）。

## 测试数据说明
1.  测试数据文件：test.txt（项目根目录下，与 src 文件夹同级）；
2.  数据格式：
    第一行：背包最大容量（整数）；
    第二行：项集数量（整数）；
    后续每行：一个项集的 6 个整数，依次为该项集 3 个物品的重量、价值（格式：w1 v1 w2 v2 w3 v3）；
3.  示例数据：已在 test.txt 中提供默认测试数据，可直接用于程序功能测试，也可自行修改数据格式进行测试。

## 注意事项
1.  运行程序前，请确保已安装所需依赖库，否则会导致绘图、Excel 保存功能无法正常使用；
2.  读取数据时，请选择 TXT 格式文件，且文件格式需符合测试数据格式要求，否则会弹出错误提示；
3.  请按界面提示顺序操作（如：需先读取数据，再执行绘图、排序、求解操作；需先求解，再执行保存操作），避免出现功能异常；
4.  保存 Excel 文件时，建议选择空文件夹保存，避免文件覆盖或路径错误。
